# Filument Foundry

## Description

Web-based 3D printing order management system featuring administrator and worker dashboards for managing orders, customer records, materials, finishes, colors, and production workflows.

Features AJAX-powered CRUD operations, role-based authentication, and activity logging. Development services are managed using Docker, with Symfony powering the application's backend architecture.

Built with Symfony, PHP, MySQL, Docker, and Tailwind CSS.
